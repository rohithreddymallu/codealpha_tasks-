# Unemployment Analysis

## Objective
Analyze unemployment trends using Python and data visualization techniques.

## Dataset
Unemployment_Rate_upto_11_2020.csv

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis
Performed data cleaning, exploratory data analysis (EDA), and visualized unemployment trends across states and over time.

## Result
Identified unemployment patterns and generated insights using visualizations.
